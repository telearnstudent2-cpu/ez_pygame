# 🏀 Stephen Curry 籃球接金幣遊戲 (Coin Collector)

這是使用 Pygame 開發的簡單趣味遊戲。玩家控制 Stephen Curry，目標是接住掉落的籃球（金幣）來得分。

## 🎮 遊戲特色

*   **主角**：像素風格的 Stephen Curry (90x90)
*   **目標**：收集掉落的籃球 (50x50)
*   **控制**：使用鍵盤左右鍵移動
*   **機制**：
    *   接住籃球：+10 分
    *   漏接籃球：-5 分

## 🛠️ 環境需求

*   Python 3.x
*   Pygame library

## 🚀 如何安裝與執行

1.  **安裝 Python**：確保您的電腦已安裝 Python。
2.  **安裝 Pygame**：
    開啟終端機或命令提示字元，輸入以下指令安裝 pygame：
    ```bash
    pip install pygame
    ```
3.  **執行遊戲**：
    在專案資料夾中，執行以下指令：
    ```bash
    python coin.py
    ```

## 🕹️ 操作說明

*   **⬅️ 左方向鍵**：向左移動
*   **➡️ 右方向鍵**：向右移動
*   **❌ 關閉視窗**：結束遊戲
